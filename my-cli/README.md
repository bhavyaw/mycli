# Usage
<!-- usage -->
```sh-session
$ npm install -g @bhavya_w/my-cli
$ my-cli COMMAND
running command...
$ my-cli (-v|--version|version)
@bhavya_w/my-cli/0.1.1 win32-x64 node-v8.10.0
$ my-cli --help [COMMAND]
USAGE
  $ my-cli COMMAND
...
```
